Hello, Lists!

When should I use a list?

Use a list when you need to store and later iterate (for x in my_items:) values in the order they were inserted.

Lists are great at...

  • Adding a value to the end (append)
  • Looking up a value using its position in the list ([0], [1], [2], etc...)

Lists are terrible at...

  • Removing a value from the beginning (pop)
  • Looking up values by their content (["a", "b", "c"].index("c"))

List Features

List Comprehensions

Create a new list in one line!

x = ["$" + str(amt) for amt in [20.36, 89.33, 90.54]]

Adding Lists Together

Put a list inside another as a sublist, or merge its values together.

x = ["a", "b", "c"]
y = ["d", "e", "f"]

x.append(y) # ["a", "b", "c", ["d", "e", "f"]]
x.extend(y) # ["a", "b", "c", "d", "e", "f"]
x = x + y   # ["a", "b", "c", "d", "e", "f"]
x = y + x   # ["d", "e", "f", "a", "b", "c"]

More List Features

Slicing

Grab sections of a list using slicing.

x = [1, 2, 3, 4, 5, 6]

x[:3]    # [1, 2, 3]
x[3:]    # [4, 5, 6]
x[1:3]   # start inclusive, end exclusive! [2, 3] 
x[::2]   # use 'step' to get odd [1, 3, 5]
x[1::2]  # combine 'start' and 'step' to get even [2, 4, 6]
x[::-1]  # reverse [6, 5, 4, 3, 2 ,1]
x[3::-1] # reverse part of list [4, 3, 2, 1]
